# Module 1: The Robotic Nervous System (ROS 2)

This module focuses on core middleware for robotic control, including ROS 2 nodes, publishers, subscribers, and services, connecting Python agents to ROS controllers via rclpy, and understanding and authoring humanoid URDF robot models.

## Topics Covered

- ROS 2 nodes, publishers, subscribers, and services
- Connecting Python agents to ROS controllers via rclpy
- Understanding and authoring humanoid URDF robot models

## Learning Outcomes

Students will be able to build a full ROS 2 control pipeline for a humanoid robot.
