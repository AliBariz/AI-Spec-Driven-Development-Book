# Module 2: The Digital Twin (Gazebo & Unity)

This module covers simulation environments and physics engines, including physics simulation fundamentals in Gazebo, high-fidelity 3D environments in Unity for human-robot interaction, and sensor simulation for LiDAR, depth cameras, and IMUs.

## Topics Covered

- Physics simulation fundamentals in Gazebo (gravity, collisions, joints)
- High-fidelity 3D environments in Unity for human-robot interaction
- Sensor simulation: LiDAR, depth cameras, IMUs

## Learning Outcomes

Students will be able to build digital twin environments for realistic humanoid testing.
